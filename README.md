# Job Tracker API

## Features

- Register new user
- Update user profile
- Create new Job (title, position, and other optional fields)
- Get all Jobs (available for a registered user)
- Get single Job
- Update a specific Job
- Delete a specific Job

## Technologies Used

- Backend: node.js, express.js
- Database: MongoDB
